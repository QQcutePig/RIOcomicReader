🚀 快速開始

Windows 前置需求: 安裝 Python 3.8+ 執行 run.bat

macOS 前置需求: 安裝 xcode-select --install 執行 start.command

Linux 啟動 前置需求: sudo apt install python3 Ubuntu/Debian sudo dnf install python3 Fedora/RHEL sudo pacman -S python Arch 執行 start.sh 啟動



一個現代化的網頁漫畫閱讀器，支援 CBZ/ZIP 檔案和圖片資料夾，具備閱讀進度追蹤、書籤管理等完整功能。

## ✨ 主要功能

### 📖 閱讀功能
- ✅ **多格式支援**：CBZ、ZIP 檔案及圖片資料夾（JPG、PNG、GIF、WebP、BMP）
- 📁 **資料夾管理**：自動掃描並整理漫畫資料夾
- 📄 **閱讀模式**：單頁/雙頁模式切換
- 🔍 **縮放模式**：符合寬度、符合高度、原始大小
- ⬅️ **閱讀方向**：從左到右 / 從右到左（支援日漫）
- 🔄 **頁面旋轉**：0°、90°、180°、270° 旋轉

### 🎯 進階功能
- 🔎 **放大鏡**：可調整尺寸的即時放大功能（小/中/大）
- 📜 **自動捲動**：可調速度（0.5x - 3.0x）
- ⏩ **自動翻頁**：設定間隔自動翻頁（3-20 秒）
- ⛶ **全螢幕模式**：沉浸式閱讀體驗
- 🌙 **深色模式**：護眼主題切換

### 💾 資料管理
- 📖 **閱讀歷史**：自動記錄每本漫畫的閱讀進度
- 🔖 **書籤系統**：每本漫畫最多 30 個書籤
- 💾 **進度同步**：匯出/匯入備份（JSON 格式）
- 📊 **統計資訊**：總漫畫數、閱讀時間、書籤數量

### 🎨 介面特色
- 🎨 **Material Icons**：專業圖標系統
- 📱 **響應式設計**：支援多種螢幕尺寸
- 🖼️ **視圖大小**：小/中/大三種網格模式
- 🔍 **搜尋排序**：依名稱、最近閱讀、進度排序

## 📖 完整使用教學

### 1️⃣ 首次使用：掃描漫畫

**支援格式：**
- `.cbz` 或 `.zip` 壓縮檔
- 圖片資料夾（`.jpg` `.png` `.gif` `.webp` `.bmp`）

**步驟：**
1. 開啟應用程式首頁
2. 點擊頂部工具列「📂 重新掃描」按鈕
3. 瀏覽器會彈出資料夾選擇視窗
4. 選擇包含漫畫的資料夾
5. 等待自動掃描完成（會顯示進度條）
6. 掃描完成後會列出所有漫畫

**資料夾結構範例：**
```
我的漫畫/
├── XXXX.cbz
├── XXXX.zip
└── 某漫畫資料夾/
    ├── 001.jpg
    ├── 002.jpg
    ├── 003.png
    └── ...
```

### 2️⃣ 開始閱讀

**開啟漫畫：**
- 點擊封面圖片開始閱讀
- 若是資料夾型漫畫，會先進入資料夾選擇要讀的漫畫

**翻頁方式（多選一）：**
- 按鍵盤 `←` `→` 箭頭
- 按鍵盤 `Space` 空白鍵
- 點擊畫面左側翻上一頁
- 點擊畫面右側翻下一頁
- 使用進度滑桿直接跳頁

**工具列功能：**（底部自動顯示，3 秒無操作會隱藏）
- ⚙️ **設定**：調整各項偏好設定
- 🔖 **書籤**：新增/移除書籤
- 🔎 **放大鏡**：啟動即時放大
- 📜 **自動捲動**：自動向下捲動
- ⏩ **自動翻頁**：自動翻頁
- 📊 **統計**：查看閱讀統計
- 📄 **頁面模式**：切換單頁/雙頁
- ⛶ **全螢幕**：進入全螢幕
- ✕ **關閉**：回到首頁

### 3️⃣ 管理書籤

**新增書籤：**
- 方法 1：在閱讀時按鍵盤 `B`
- 方法 2：點擊工具列的 🔖 圖示

**查看書籤：**
1. 回到首頁
2. 點擊工具列「🔖 書籤」
3. 會列出所有書籤（漫畫名稱 + 頁數）
4. 點擊書籤項目快速跳轉到該頁

**刪除書籤：**
- 在書籤清單中點擊每個項目旁的 🗑️ 按鈕

**書籤限制：**
- 最多 30 個書籤
- 超過時會有提示

### 4️⃣ 查看閱讀歷史

1. 回到首頁
2. 點擊工具列「📜 歷史」
3. 會顯示最近 6 本閱讀的漫畫
4. 功能：
   - **繼續閱讀**：從上次位置繼續
   - **刪除**：移除該漫畫的歷史記錄

### 5️⃣ 備份與還原

**匯出備份：**
1. 回到首頁
2. 點擊「⚙️ 設定」
3. 切換到「💾 備份」分頁
4. 點擊「📤 匯出備份」
5. 會下載 `comic-backup-YYYY-MM-DD.json` 檔案

**匯入備份：**
1. 點擊「📥 匯入備份」
2. 選擇之前匯出的 `.json` 檔案
3. 會**覆蓋**現有的所有進度和書籤

**備份包含內容：**
- ✅ 所有漫畫的閱讀進度
- ✅ 所有書籤資料
- ✅ 匯出時間戳記
- ❌ 漫畫檔案本身不會備份

### 6️⃣ 個人化設定

**顯示設定分頁：**
- 📄 **頁面模式**：單頁 / 雙頁
- 📐 **縮放模式**：符合寬度 / 符合高度 / 原始大小
- 🔄 **旋轉角度**：0° / 90° / 180° / 270°
- 🔎 **放大鏡尺寸**：小(120x180) / 中(160x240) / 大(200x300)

**操作設定分頁：**
- 📖 **閱讀方向**：從左到右 / 從右到左（適合日漫）
- 📜 **自動捲動速度**：0.5x - 3.0x（按 +/- 調速）
- ⏱️ **自動翻頁間隔**：3 / 5 / 8 / 10 / 15 / 20 秒

**備份設定分頁：**
- 📤 **匯出備份**：下載進度和書籤
- 📥 **匯入備份**：還原之前的備份

| 瀏覽器 | 版本 | 支援狀態 | 備註 |
|--------|------|---------|------|
| Chrome | 86+ | ✅ **完全支援** | **推薦使用** |
| Edge | 86+ | ✅ **完全支援** | **推薦使用** |

**為什麼某些瀏覽器不支援？**
- 本應用使用 `File System Access API` 讀取本機檔案
- 目前只有 Chromium 核心瀏覽器完全支援
- Firefox 需要在設定中手動啟用實驗功能

## ❓ 常見問題

### Q: 資料會上傳到網路嗎？
**A:** 完全不會！所有資料（漫畫、進度、書籤）都儲存在**你的瀏覽器本地**（LocalStorage/IndexedDB），完全離線運作。GitHub 只託管應用程式本身，不會上傳任何個人資料。

### Q: 為什麼 Firefox 無法掃描資料夾？
**A:** Firefox 尚未完全支援 File System Access API。建議使用 Chrome 或 Edge 獲得最佳體驗。

### Q: 可以在手機上使用嗎？
**A:** 理論上支援（手機版 Chrome/Edge），但檔案選擇器可能受系統限制。建議在電腦上使用以獲得最佳體驗。

### Q: 換電腦後如何轉移進度？
**A:** 
1. 在舊電腦上「匯出備份」下載 JSON 檔案
2. 在新電腦上開啟應用程式
3. 進入「設定」→「匯入備份」
4. 選擇剛才的 JSON 檔案
5. 所有進度和書籤會恢復

### Q: 閱讀進度會自動儲存嗎？
**A:** 會！每次翻頁都會自動儲存當前進度。關閉瀏覽器後再開啟，會從上次位置繼續。

### Q: 封面圖沒有顯示？
**A:** 封面會在背景自動提取，請稍等片刻。如長時間未顯示，可嘗試：
1. 重新整理頁面（按 F5）
2. 重新掃描資料夾
3. 檢查漫畫檔案是否正確

### Q: 可以刪除漫畫或進度嗎？
**A:** 
- **刪除進度**：在歷史清單中點擊 🗑️ 按鈕
- **刪除漫畫**：在檔案系統中刪除漫畫檔案，然後重新掃描

### Q: 支援繁體中文嗎？
**A:** 是的！完全支援繁體中文，包括搜尋、排序、介面文字等。

### Q: 這是免費軟體嗎？
**A:** 是的！採用 MIT License，完全免費開源，可自由商用、修改、發布。
